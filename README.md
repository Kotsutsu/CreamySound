# 🎧 CreamySound

> BetterDiscord plugin that adds keyboard typing sounds like Opera GX directly into Discord.

---

## ✨ Features

- 🔊 Dynamic keyboard sounds
- 🎲 Random sounds for letter keys
- ⌨️ Dedicated sounds for:
  - Enter
  - Backspace
  - Space
- ⚡ Lightweight and optimized (anti-spam cooldown)
- 🌐 Sounds loaded directly from GitHub (no local files required)

---

## 📦 Installation

1. Download `CreamySound.plugin.js`
2. Place it in your BetterDiscord plugins folder:


C:\Users\YOUR_NAME\AppData\Roaming\BetterDiscord\plugins


3. Open Discord
4. Go to **Settings → Plugins**
5. Enable **CreamySound**
6. (Optional) Press `Ctrl + R` if needed
